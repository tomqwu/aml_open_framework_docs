Act II · The Spec Is The Contract

Two layers of validation.

The Manifest doesn't load if either layer fails. Structure first, then cross-references. Both run before the engine touches a row of data.

Layer 1 Structural validation JSON Schema · Draft 2020-12
Shape, types, required fields. Catches what's missing or wrongly typed before any cross-reference is checked.
  • Required field absent — rules[2].id missing
  • Wrong type — severity must be one of "low" / "medium" / "high"
  • Unknown field — rules[*].extra: forbid flags typos
  • Discriminator — logic.type must be one of the 4 RuleLogic union members
Example error · CLI exit 1 SchemaError: rules[2].severity expected one of ["low","medium","high"] got "critical"
Layer 2 Cross-reference + types Pydantic v2 · extra="forbid"
References, lifecycle, semantics. Catches what only makes sense once the whole document is in scope.
  • Dangling regulation_ref — rule cites "PCMLTFA s.99" not in the regulation index
  • Unknown queue — escalate_to points to a queue not declared in workflow.queues
  • Threshold sanity — gte is greater than lte on the same metric
  • SLA on a non-existent severity — sla_by_severity references a tier the program doesn't define
Example error · CLI exit 1 CrossRefError: rules[0].escalate_to references queue "l1_aml_analyst" not declared in workflow.queues
Two layers before a single row of production data is touched.
