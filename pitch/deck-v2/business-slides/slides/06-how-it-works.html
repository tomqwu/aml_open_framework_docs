Exhibit 6 How it works

Five stages from one document to a regulator-ready audit.

One Manifest. Five deterministic transformations. Every artefact in the chain — alerts, cases, STRs, audit ledger entries — generated from the document, never hand-written in parallel.

📜
Stage · 01
Compliance Manifest
Compliance writes one document. Every rule cites the regulation that justifies it. PR-reviewed, version-controlled.

Stage · 05
Audit ledger
Every decision SHA-256-hash-chained. Re-run any history byte-for-byte. Tampering breaks the chain.
Verified by · 2LoD / Audit
Determinism guarantee same Manifest + same data + same seed = identical output hashes. No commercial platform publishes this guarantee. SR 26-2 expects it from challenger models.
